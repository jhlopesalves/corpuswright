# Public Domain Demo Corpus

This corpus is a completely synthetic set of files created strictly for testing and demonstrating CorpusWright's cleaning parameters.

**License**: Public Domain (CC0)
**Source**: Synthetically generated random text and formatting artifacts.

**Notes**:
- The `clean/` directory contains files with standard formatting.
- The `dirty/` directory contains files with intentionally injected formatting artifacts (e.g., page numbers, inconsistent newlines, Roman numerals, page delimiters, HTML tags).
- These files are not intended for linguistic research. They are purely mechanical test files.
