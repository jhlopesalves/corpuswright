**Modifications**:
- The books have been stripped of their standard Project Gutenberg headers and footers to the best extent possible automatically.
- Each book has been artificially chunked into roughly 5-10 smaller files to keep file sizes manageable for demonstration and automated testing purposes.
- Realistic artifacts such as standalone page numbers (e.g. `Page 42`, `--- 42 ---`) and Roman numerals have been occasionally injected between paragraphs to demonstrate CorpusWright's cleaning parameters against authenticly noisy historical OCR/scan data.
